## 🤖 **AI 멘토링 시스템**

| 역할 | AI | 설명 |
|------|----|------|
| **메인 멘토** | Claude (로드) | 질문-토론 중심, 깊이 있는 학습, 전략 설계 |
| **보조 튜터** | GPT (피티) | 디버깅, 코드 생성, 기술적 지원 |

### 특별한 학습 방식
- **복붙 0% 원칙**: 모든 코드를 직접 작성하여 진짜 실력 키우기
- **단계별 학습**: 개념 → 예제 → 실습 → 이해도 체크
- **실무 연결**: 엣지케이스 처리, 확장성 고려한 설계
